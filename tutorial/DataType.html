Js DataTypes

-->JavaScript has totally 4 data types.
-->JavaScript Number :JavaScript has only one type of numbers.
-->JavaScript Boolean :Booleans can only have two values: true or false.
-->JavaScript String : Strings are written with quotes. You can use single or double quotes:
-->JavaScript Array :JavaScript arrays are written with square brackets.
-->JavaScript Object:Object properties are written as name:value pairs, separated by commas.
